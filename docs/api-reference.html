Authentication

POST /auth/login

Authenticate admin user and receive JWT token.

Request Body:

{ "email": "admin@jesus-travel.com", "password": "JesusWayne6667" }

Response (200 OK):

{ "success": true, "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...", "user": { "email": "admin@jesus-travel.com", "role": "admin" } }

Error Response (401 Unauthorized):

{ "success": false, "error": "Invalid credentials" }
üîê Protected Routes: All routes below require authentication. Include the JWT token in the Authorization header:
Authorization: Bearer YOUR_JWT_TOKEN

Enquiries

POST /enquiries

Create a new booking enquiry (public endpoint - no auth required).

Request Body:

{ "customerName": "John Smith", "customerPhone": "+447700123456", "customerEmail": "john@example.com", "pickupLocation": "Heathrow Airport", "dropoffLocation": "Central London", "pickupDate": "2025-01-15", "pickupTime": "14:00", "passengers": 2, "vehicleType": "Saloon", "specialRequests": "Meet & greet required" }

Response (201 Created):

{ "success": true, "data": { "id": "abc123", "referenceNumber": "JT-2025-000123", "status": "pending_quote", "customerName": "John Smith", ... } }

GET /enquiries

Get all enquiries (requires authentication).

Query Parameters:

Parameter Type Description
status string Filter by status (pending_quote, quoted, confirmed, etc.)
limit number Number of results (default: 50)
offset number Pagination offset (default: 0)

Response (200 OK):

{ "success": true, "data": [ { "id": "abc123", "referenceNumber": "JT-2025-000123", "status": "pending_quote", "customerName": "John Smith", "customerPhone": "+447700123456", "pickupLocation": "Heathrow Airport", "dropoffLocation": "Central London", "pickupDate": "2025-01-15", "pickupTime": "14:00", "createdAt": "2025-01-10T10:30:00Z" } ], "total": 1 }

GET /enquiries/:id

Get a specific enquiry by ID.

Response (200 OK):

{ "success": true, "data": { "id": "abc123", "referenceNumber": "JT-2025-000123", "status": "quoted", "quote": { "price": 150, "currency": "GBP", "notes": "Includes meet & greet", "validUntil": "2025-01-17T14:00:00Z" }, ... } }

PUT /enquiries/:id/quote

Submit a quote for an enquiry (requires authentication).

Request Body:

{ "price": 150, "currency": "GBP", "notes": "Includes meet & greet and 30 mins waiting time", "validUntil": "2025-01-17T14:00:00Z", "breakdown": { "basePrice": 120, "meetGreet": 20, "waitingTime": 10 } }

Response (200 OK):

{ "success": true, "data": { "id": "abc123", "status": "quoted", "quote": { "price": 150, "currency": "GBP", "notes": "Includes meet & greet and 30 mins waiting time", "validUntil": "2025-01-17T14:00:00Z" } }, "message": "Quote sent successfully" }

PUT /enquiries/:id/forward-to-partner

Forward an enquiry to a partner company.

Request Body:

{ "partnerName": "ABC Chauffeurs", "commissionRate": 15, "bookingReference": "ABC-12345", "notes": "Customer prefers luxury vehicle" }

Response (200 OK):

{ "success": true, "data": { "id": "abc123", "status": "forwarded_to_partner", "partnerInfo": { "partnerName": "ABC Chauffeurs", "commissionRate": 15, "bookingReference": "ABC-12345" } } }

DELETE /enquiries/:id

Delete an enquiry (requires authentication).

Response (200 OK):

{ "success": true, "message": "Enquiry deleted successfully" }

DELETE /enquiries/clear-all

Delete all enquiries (requires authentication). Use with caution!

Response (200 OK):

{ "success": true, "message": "All enquiries cleared", "deletedCount": 42 }

Status Codes

Code Meaning Description
200 OK Request successful
201 Created Resource created successfully
400 Bad Request Invalid request data
401 Unauthorized Authentication required or failed
404 Not Found Resource not found
500 Internal Server Error Server error occurred

Enquiry Status Values

Status Description
pending_quote Enquiry received, waiting for quote
quoted Quote sent to customer
confirmed Customer accepted quote
cancelled Booking cancelled
completed Service completed
forwarded_to_partner Forwarded to partner company

Example: Creating an Enquiry with JavaScript

const createEnquiry = async () => { try { const response = await fetch('https://jesus-travel-back.onrender.com/api/enquiries', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ customerName: 'John Smith', customerPhone: '+447700123456', customerEmail: 'john@example.com', pickupLocation: 'Heathrow Airport', dropoffLocation: 'Central London', pickupDate: '2025-01-15', pickupTime: '14:00', passengers: 2, vehicleType: 'Saloon' }) }); const data = await response.json(); if (data.success) { console.log('Enquiry created:', data.data.referenceNumber); } else { console.error('Error:', data.error); } } catch (error) { console.error('Network error:', error); } };

Example: Authenticated Request

const getEnquiries = async (token) => { try { const response = await fetch('https://jesus-travel-back.onrender.com/api/enquiries', { method: 'GET', headers: { 'Authorization': `Bearer ${token}`, 'Content-Type': 'application/json' } }); const data = await response.json(); if (data.success) { console.log('Enquiries:', data.data); } else { console.error('Error:', data.error); } } catch (error) { console.error('Network error:', error); } };
