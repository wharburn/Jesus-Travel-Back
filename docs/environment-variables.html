üîß Server Configuration

NODE_ENV REQUIRED

Value: production

Description: Sets the application environment mode. Must be "production" for Render deployment.

PORT REQUIRED

Value: 3000

Description: Port number for the server. Render automatically sets this, but 3000 is the default.

FRONTEND_URL REQUIRED

Value: https://jesus-travel.site

Description: Your frontend website URL for CORS configuration.

üíæ Database Configuration (Upstash Redis)

UPSTASH_REDIS_REST_URL REQUIRED

Example: https://light-pika-5042.upstash.io

Description: Upstash Redis REST API URL. Get this from your Upstash dashboard.

How to get:

  1. Go to console.upstash.com
  2. Select your Redis database
  3. Copy the "REST URL" from the dashboard

UPSTASH_REDIS_REST_TOKEN REQUIRED

Example: AYQxASQgNTk...

Description: Upstash Redis authentication token.

How to get: Same location as REST URL in Upstash dashboard - copy "REST TOKEN"

üß† OpenRouter AI

OPENROUTER_API_KEY REQUIRED

Example: sk-or-v1-abc123...

Description: OpenRouter API key for AI-powered responses.

How to get:

  1. Go to openrouter.ai
  2. Sign up / Log in
  3. Go to Keys section
  4. Create new API key

OPENROUTER_MODEL OPTIONAL

Default: anthropic/claude-3.5-sonnet

Description: AI model to use. Defaults to Claude 3.5 Sonnet if not specified.

üì± WhatsApp Configuration (Green API)

GREEN_API_INSTANCE_ID REQUIRED

Example: 7103123456

Description: Your Green API WhatsApp instance ID.

How to get:

  1. Go to green-api.com
  2. Sign up and create an instance
  3. Copy the Instance ID from dashboard
  4. Scan QR code to connect WhatsApp

GREEN_API_TOKEN REQUIRED

Example: abc123def456...

Description: Green API authentication token for your instance.

How to get: Found in the same location as Instance ID in Green API dashboard

üîê Admin Authentication

ADMIN_JWT_SECRET REQUIRED

Example: your-super-secret-jwt-key-change-this

Description: Secret key for signing JWT tokens. Must be a strong random string.

How to generate:

openssl rand -base64 32

Run this command in terminal to generate a secure random key.

ADMIN_DEFAULT_EMAIL REQUIRED

Value: admin@jesus-travel.com

Description: Default admin email for login.

ADMIN_DEFAULT_PASSWORD REQUIRED
